功能概述

了解孝心圈的核心功能，让关爱更便捷

每日打卡

每日打卡是孝心圈的核心功能，让老人可以通过简单的方式告诉家人自己一切安好。

  • ✓ 点击打卡 - 最简单的方式，轻轻一点即可
  • ✓ 声音打卡 - 录制语音，让家人听到你的声音
  • ✓ 计步打卡 - 自动检测运动，记录健康状态
📱
📊

健康记录

完整的打卡历史记录，帮助家人了解老人的日常健康状况。

  • ✓ 历史记录 - 查看所有打卡记录
  • ✓ 统计分析 - 分析打卡频率和规律
  • ✓ 健康趋势 - 追踪长期健康变化
  • ✓ 数据导出 - 可导出健康数据

消息通知

及时的消息提醒，让贴心人不错过任何重要的关爱时刻。

  • ✓ 打卡提醒 - 温馨提醒打卡
  • ✓ 异常提醒 - 长时间未打卡时提醒贴心人
  • ✓ 消息推送 - 即时推送重要信息
  • ✓ 自定义提醒 - 自定义提醒时间和内容
💬
🏃

智能运动检测

利用手机加速度计，自动检测日常活动，智能记录健康状态。

  • ✓ 自动检测 - 无需手动操作，自动检测活动
  • ✓ 智能判断 - 识别日常运动模式
  • ✓ 隐私保护 - 数据本地处理，不上传个人信息
  • ✓ 省电优化 - 低功耗算法，不影响电池寿命
